# معايير المقارنة

#### التقنية

1. المنصة (ويب/موبايل)
2. الإطار/الأطر المستخدمة
3. مشغل الصوت
4. عرض النص، الصور، أو SVG
5. بدون إنترنت أو مع إنترنت
6. ميزات الذكاء الاصطناعي
7. الترخيص
8. الوضع الداكن

2. تخطيطات المصحف المدعومة
3. القراءات المدعومة
4. الترجمات المدعومة
5. التفاسير المدعومة
6. البحث النصي الكامل
7. البحث الدلالي

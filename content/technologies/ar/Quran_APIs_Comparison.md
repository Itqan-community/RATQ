# مقارنة واجهات برمجة تطبيقات القرآن الكريم (Quran APIs)

توفر هذه الوثيقة دليلاً شاملًا للمطورين الراغبين في بناء تطبيقات قرآنية، حيث تستعرض وتقارن بين أشهر واجهات برمجة التطبيقات (APIs) المتاحة حاليًا. يهدف هذا الدليل إلى مساعدتك في اختيار الأداة الأنسب لمشروعك بناءً على معايير فنية دقيقة مثل: حدود الاستخدام (Rate Limits)، المميزات (Features)، وتنسيق البيانات (Data Formats).

## مصفوفة المقارنة

| اسم الواجهة (API Name) | حدود الاستخدام (Rate Limits) | المميزات والخصائص (Features) | تنسيق البيانات (Format) | المصادقة (Auth) | التوثيق (Docs) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Quran.com API (V4)** | غير محدد / مفتوح (الاستخدام العادل) | • نص القرآن (رسم عثماني، إملائي، إندوباك)<br>• التلاوات الصوتية<br>• التفاسير والترجمات (+100)<br>• التحليل الصرفي (Word-by-word)<br>• بحث متقدم | JSON | يتطلب مفتاح وصول (Client Credentials) للوصول للمحتوى الكامل. | [رابط](https://api-docs.quran.foundation/) |
| **Al Quran Cloud** | ~10 طلبات/ثانية<br>تراكمي: 70 طلب/7 ثوانٍ | • النصوص بإصدارات متعددة<br>• الصوتيات (سورة وآية)<br>• البيانات الوصفية (أحزاب، أجزاء، سجدات)<br>• صور الآيات وتجويد | JSON | غير مطلوب. | [رابط](https://alquran.cloud/api) |
| **QuranEnc** | حدود تعتمد على مستوى الاشتراك | • **الترجمات الموثوقة** (الميزة الأساسية)<br>• تصفح حسب السورة/الآية/الصفحة<br>• التفاسير المعتمدة | JSON | **مطلوب مفتاح API** (للإصدار v2) | [رابط](https://quranenc.com/en/home#api) |
| **Alfanous** | غير محدد / متاح للاستضافة الذاتية | • **محرك بحث دلالي متقدم**<br>• البحث بالجذر والمشتقات<br>• التحليل اللغوي والصرفي | JSON | غير مطلوب (للنسخة العامة). | [رابط](https://www.alfanous.org/api/) |
| **EveryAyah** | غير محدد (الاستخدام العادل) | • **مكتبة صوتية ضخمة** (آية بآية)<br>• توفر أكثر من 30 قارئًا<br>• تقسيم الآيات صوتيًا | JSON, MP3 | غير مطلوب. | [رابط](https://everyayah.com/data/status.php) |
| **QuranHub** | غير محدد (أداء عالٍ عبر Cloudflare) | • سرعة استجابة عالية (Edge Caching)<br>• نص القرآن والترجمات<br>• واجهة حديثة وسريعة | JSON | غير مطلوب. | [رابط](https://github.com/misraj-ai/quranhub) |

## التوصيات والاستخدام الأمثل

- **للتطبيقات العامة الشاملة:** يُنصح باستخدام **Quran.com API** أو **Al Quran Cloud** نظرًا لشموليتهما (نص، صوت، ترجمة) واستقرارهما.
- **للبحث المتقدم والدلالي:** يُعد **Alfanous** الخيار الأفضل بلا منازع للتطبيقات التي تتطلب تحليلًا لغويًا وبحثًا في المعاني والجذور.
- **للتطبيقات الصوتية التعليمية:** يوفر **EveryAyah** مرونة عالية في التعامل مع الملفات الصوتية على مستوى الآية، مما يجعله مثاليًا لتطبيقات التحفيظ.
- **للترجمات المعتمدة:** إذا كان تركيز تطبيقك على دقة الترجمات وموثوقيتها، فإن **QuranEnc** هو المصدر والمرجع المعتمد في هذا المجال.

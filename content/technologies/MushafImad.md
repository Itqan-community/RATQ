# Mushaf Imad

[MushafImad](https://github.com/ibo2001/MushafImad) is a Swift Package that delivers a fully featured Mushaf (Quran) reading experience for iOS 17+ and macOS 14+.
So, as a quran app developer, you only need to focus on implementing your idea without worrying about the UI and the Quran data.
You use the package as a dependency in your app, and customize it to fit your application.

## Features

The package ships :

1. page images
2. verse metadata
3. timing information
4. audio helpers
5. polished SwiftUI components
6. audio playback
7. toast feedback
8. contextual navigation

> For more information, you should visit the repository, it has a comprehensive documentation that covers all the features and how to use them.

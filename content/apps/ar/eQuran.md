# eQuran

تطبيق هاتف محمول لكل مستخدم.

> تنويه: مصدر التطبيق تمت أرشفته بواسطة المالك ولكن يمكنك الاطلاع عليه على github حيث أنه أصبح للإطلاع فقط

## المميزات

- عرض القرآن الكريم كاملاً حسب السورة، الجزء، أو المفضلة.
- إمكانية إضافة السور إلى قائمة المفضلة.
- البحث عن طريق اسم السورة أو رقمها لضمان التصفية السريعة.
- الانتقال السريع إلى آخر آية تمت قراءتها.
- تصفح السور يتم بنظام "آية بآية"، مع عرض تنسيق الآية والترجمة أسفلها مباشرة.
- تشغيل تلاوة للآية (بصوت الشيخ مشاري العفاسي)، وتتطلب هذه الميزة اتصالاً بالإنترنت.
- أثناء التصفح، يمكنك الانتقال إلى آية محددة (عبر أيقونة النقاط الثلاث في الأعلى).
- في الإعدادات، يمكنك تخصيص التجربة كما يلي:
    - تشغيل أو إيقاف التغذية الراجعة اللمسية (Haptic Feedback) عند التنقل.
    - عرض كل آية بشكل منفصل، أو عرض الكل في صفحة واحدة (مع بقاء كل آية في بطاقة مستقلة، لكنك تختار عرض آية واحدة أو كل آيات السورة في صفحة واحدة).
    - تحديد عدد نتائج "آخر قراءة" ليظهر نتيجة واحدة أو ما يصل إلى 7 نتائج.
    - اختيار الترجمة المناسبة من بين 8 ترجمات متاحة.
    - تخصيص سرعة تشغيل التلاوة.
    - اختيار مظهر الألوان (Theme) المفضل لديك.
    - تعديل حجم الخط.
    - مسح سجل القراءة أو قائمة المفضلة.

## للمطورين

هذا تطبيق للهواتف المحمولة مفتوح المصدر مكتوب باستخدام إطار عمل Flutter. يمكنك العثور على الكود المصدري هنا: [https://github.com/ya27hw/equran_app](https://github.com/ya27hw/equran_app).
مصدر البيانات هو ملف json موجود في `equran_app/blob/main/assets/index.json` والذي يسرد جميع سور القرآن بالشكل التالي:
```json
[
    {
        "id": 1,
        "name": "الفاتحة",
        "transliteration": "Al-Fatihah",
        "type": "meccan",
        "total_verses": 7,
        "link": "https://cdn.jsdelivr.net/npm/quran-json@3.1.2/dist/chapters/1.json"
    }
]        
```
مصدر هذا الملف هو [quran-json](../../resources/ar/quran-json.md) والذي يعيد هذا الملف كاستجابة (Response).

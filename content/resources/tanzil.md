# Tanzil

## Overview

 Tanzil is a Quranic project launched in 2007 to produce a highly verified Unicode Quran text to be used in Quranic websites and applications. Tanzil project is made to produce a standard Unicode Quran text and serve as a reliable source for this standard text on the web. Tanzil provides a standard Unicode Quran text in various scripts (Uthmani, Simple Clean) and translations.

## Features

- **Accuracy**

    Accuracy is the most important factor in Tanzil. The text provided by Tanzil project is the most reliable and precise digital Quran text available on the web.

- **Searchability**

    Both Uthmani and Imlaei versions of Tanzil Quran text are completely searchable and can be easily embedded in other text formats.

- **Pause Marks**

    The Imlaei Quran text provided by Tanzil is the first such text that includes complete pause marks.

- **Compatibility**

    Our Uthmani Quran text is completely matching the Medina Mushaf. The writing style of our Imlaei text also has a perfect match to the standard Mushaf.

- **Flexibility**

    Tanzil provides options for including or removing diacritics, pause marks, different tanween shapes, etc., to better suit different usages and various fonts available.


## Quran Navigator

- Tanzil provides a navigator that allows the users to browser the quran by displaying the quran in user's preferred script (Uthmani, Imlaei), its tafsir, and its translations.

- It also, provides the ability to search the Quran text, translations, and recitations by word or root of the word.

- It also provides recitation verse-by-verse.

- You can change the font.

You can find the Quran Navigator here: [https://tanzil.net/](https://tanzil.net/)


## Resources for Developers

Tanzil provides many resources to help the developers of Quranic websites and applications. You can find the resources here:

1. **Quran Text**

    Tanzil provides the Quran text in various scripts (Uthmani, Imlaei), also in different formats (text, text with aya number, XML, SQL (MySql dump)).
    You can find the resources here: [https://tanzil.net/download/](https://tanzil.net/download/)

2. **Quran Fonts**

    Tanzil provides the Quran fonts:

    - **KFGQPC Hafs Uthmanic Script**: King Fahd Quran Complex
    - **KFGQPC Uthman Taha Naskh**: King Fahd Quran Complex
    - **Scheherazade**: SIL International
    - **me_quran**: Meor Ridzuan
    - **PDMS Saleem Quran Font**: Pakistan Data Management Services

    You can find the resources here: [https://tanzil.net/docs/quranic_fonts](https://tanzil.net/docs/quranic_fonts)

3. **Quran Metadata**

    Also, you can download the Quran metadata in XML and JavaScript formats from this link: [https://tanzil.net/docs/quran_metadata](https://tanzil.net/docs/quran_metadata)

    - `quran-data.xml`: Metadata in XML
    - `quran-data.js`: A compressed version of metadata in JavaScript.

4. **Quran verse-by-verse recitations**

    For this part, tanzil provide a link to [https://www.versebyversequran.com/](https://www.versebyversequran.com/) which is another domain for the same server of [https://everyayah.com/](https://everyayah.com/).

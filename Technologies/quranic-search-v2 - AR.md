# مراجعة المشروع: Quranic Search v2

**تاريخ المراجعة:** 11/11/2025

**نوع المشروع:** محرك بحث دلالي للنص العربي (القرآن)

**رابط المشروع:** [https://github.com/ahr9n/quranic-search-v2](https://github.com/ahr9n/quranic-search-v2)

**المكدس التقني:** React.js, Django, Flask, Gensim (Word2Vec), SQLite

---

## الملخص التنفيذي

Quranic Search v2 هو تطبيق ويب يوفر قدرات البحث المعجمي (القائم على الكلمات المفتاحية) والدلالي (القائم على المعنى) للقرآن الكريم. يستخدم المشروع تقنيات معالجة اللغة الطبيعية (NLP)، وتحديداً تضمينات Word2Vec، لتمكين البحث الدلالي في النص العربي. تتكون البنية من واجهة أمامية React، وواجهة برمجية Django للبحث المعجمي، وواجهة برمجية Flask للبحث الدلالي.

---

## هيكل المشروع

### نظرة عامة على البنية

```bash
                     → Lexical API (Django) → SQLite Database
Frontend (React.js) →
                     → Semantic API (Flask) → Word2Vec Models → Lexical API
```

### المكونات

1. **الواجهة الأمامية** (`frontend/`)
   - تطبيق React.js مع React Router
   - المكونات: HomeForm, ResultsForm, Verse, Navbar
   - الحاويات: Home, Results, About, Bookmarks

2. **واجهة البحث المعجمي** (`backend/api/lexical/`)
   - Django REST Framework
   - قاعدة بيانات SQLite مع آيات القرآن
   - نقاط النهاية لاسترجاع الآيات والبحث بالكلمات المفتاحية

3. **واجهة البحث الدلالي** (`backend/api/semantic/`)
   - واجهة برمجية RESTful باستخدام Flask
   - نماذج Word2Vec (نموذج AraVec Twitter)
   - طرق متعددة لتقييم التشابه (max, frequency, average, pooling)

### نقاط القوة

1. **فصل واضح للاهتمامات**
   - الواجهة الأمامية، واجهة البحث المعجمي، وواجهة البحث الدلالي منفصلة بشكل جيد
   - هيكل كود معياري مع مجلدات مخصصة

2. **توثيق شامل**
   - README مفصل مع نظرة عامة على المشروع
   - تعليقات في الكود في الطرق الدلالية تشرح الخوارزميات
   - توثيق هيكل المجلدات

3. **طرق بحث متعددة**
   - ينفذ طرق تقييم تشابه متنوعة
   - يجمع نماذج وطرق متعددة للحصول على نتائج أفضل

4. **مكدس تقني حديث**
   - React 18.2.0

## اعتبارات الأداء

1. **تحميل النماذج**
   - نماذج Word2Vec تُحمّل عند استيراد الوحدة
   - النماذج تبقى في الذاكرة لجميع الطلبات
   - **التأثير:** استخدام عالي للذاكرة، لكن جيد لوقت الاستجابة

2. **استعلامات قاعدة البيانات**
   - Django ORM مستخدم بشكل مناسب
   - لا توجد مشاكل N+1 واضحة

3. **تحسين الواجهة الأمامية**
   - يستخدم React 18 (جيد)
   - لا يوجد تقسيم كود أو تحميل كسول واضح

## الخلاصة

Quranic Search v2 هو مشروع منظم بشكل جيد مع هدف واضح وفصل جيد للاهتمامات. تنفيذ البحث الدلالي باستخدام Word2Vec مثير للاهتمام ومناسب لحالة الاستخدام. ومع ذلك، هناك عدة مشاكل تحتاج إلى معالجة، خاصة حول إدارة التبعيات، متطلبات إصدار Python، ومعالجة الأخطاء.

## قسم ملاحظات المراجع

1. بعد تثبيت جميع التبعيات، وإصلاح أي تعارضات مع التبعيات، عمل المشروع بنجاح.
   > النموذج غير متضمن في المستودع، لذا ستحمّله كجزء من إعداد المشروع وهو كبير جداً (حوالي 1.2GB).
2. تم اختبار البحث المعجمي والدلالي، وكلاهما عمل بنجاح.
3. بالنسبة للبحث المعجمي، كانت النتائج كما هو متوقع.
4. بالنسبة للبحث الدلالي، لم تكن النتائج دقيقة جداً. النتائج الأولى أكثر ارتباطاً بالاستعلام، لكن بعد ذلك، تتباعد النتائج بشكل كبير عن الاستعلام.

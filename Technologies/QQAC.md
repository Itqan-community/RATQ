# Quran Question Answer Context

## Overview

[QQAC](https://huggingface.co/datasets/nazimali/quran-question-answer-context) is huggingface dataset containing 1224 annotated question-answers in both arabic and english.
It provides translation to the original dataset [AQQAC](./AQQAC.md) from arabic to english, and adds the Surah ayahs to the context column.

## Key Features

- 1224 annotated question-answers.
- Available for usage in huggingface datasets, or importing using python library `datasets`.
- The questions are available in Arabic and English.

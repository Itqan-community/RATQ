# Annotated Corpus of Arabic Al-Quran Question and Answer

## Overview

[AQQAC](https://archive.researchdata.leeds.ac.uk/464/) is a collection of approximately 2224 questions and answers about Al-Al-Quran with 1224 annotated question-answers available for download and public use (the rest is due to copyright concerns).
Each question and answer is annotated with the question ID, question word (particles), chapter number, verse number, question topic, question type, Al-Quran ontology concepts (Alqahtani & Atwell, 2018) and question source.

## Key Features

- 1224 annotated question-answers.
- Available for download in both xml and xlsx formats.
- The questions are available in Arabic only.
